# SpotifyClone
This project is a Spotify-inspired music streaming clone built using pure HTML, CSS, and JavaScript. It focuses on creating a clean, modern UI similar to Spotify while maintaining simplicity and beginner-friendly code. It has a responsive design (desktop &amp; mobile friendly).

Features : 
1. Music cards with audio playback
2. Spotify-style dark theme
3. Responsive design (desktop & mobile friendly)
4. Smooth scroll navigation
5. Scroll-to-top button
6. About Us section
7. Contact Us form
8. Clean and organized code structure

Tech Stack :
HTML5
CSS3
JavaScript

Purpose :
This project was built to practice frontend web development, layout design, responsiveness, and UI styling inspired by real-world applications like Spotify.
